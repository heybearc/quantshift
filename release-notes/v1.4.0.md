# What's New in Version 1.4.0

**Released:** February 4, 2026

---

## ‚ú® New Features

### Enhanced Trading Metrics Dashboard

Your dashboard now shows more detailed trading performance metrics to help you track your success.

**What you'll see:**

- **Win Rate Card** - See your winning trades vs. losing trades at a glance, with clear percentage and ratio display
- **Max Drawdown Card** - Monitor your largest portfolio decline with color-coded severity indicators (green = healthy, yellow = caution, red = attention needed)
- **Strategy Performance Card** - View your current trading strategy and its success rate

**What you can do:**

- Quickly assess your trading performance without digging through reports
- Identify when your portfolio needs attention with visual indicators
- Track which strategies are working best for you

### Admin Statistics Dashboard (Admin Users Only)

Administrators now have powerful new tools to monitor system health and user activity.

**What you'll see:**

- **User Statistics** - Total users, active users, pending invitations, and inactive accounts
- **Session Management** - Current active sessions, peak sessions, and average session duration
- **Audit Activity** - 24-hour event count, critical events, and warnings
- **System Health** - Overall system status, API response times, and database connection pool status

**What you can do:**

- Monitor system performance in real-time
- Track user engagement and activity patterns
- Identify potential issues before they become problems
- Make data-driven decisions about system resources

### Visual Trend Indicators

All dashboard metrics now show trend information to help you understand changes over time.

**What you'll see:**

- **Trend Arrows** - Up/down arrows showing if metrics are improving or declining
- **Percentage Changes** - Clear percentage indicators showing the magnitude of changes
- **Color-Coded Badges** - Green for positive trends, red for negative trends

## üîí Security & Stability Improvements

### Session Management Enhancement

We've implemented industry-standard session management to improve security and system performance.

**What changed:**

- Each user can now have a maximum of 3 active sessions at a time
- When you log in on a 4th device, your oldest session is automatically logged out
- This prevents session buildup and improves system security

**What you need to know:**

- You won't notice any difference in normal usage
- If you use multiple devices, you may be logged out of older sessions automatically
- This is a security best practice used by major platforms
